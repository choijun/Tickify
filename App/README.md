## Instalation 

1. Clone repository
```shell
git clone https://github.com/pierreguerin0341/Tickify
```
2. Get in the project folder
```shell
cd app
```
3. Install dependencies via npm or yarn
```shell
npm install
```
```shell
yarn install
```

## Quick start
Run development server
```shell
yarn run serve
```
